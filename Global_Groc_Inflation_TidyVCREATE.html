Introduction

This document will analyze Global Grocery Inflation data to demonstrate select packages within Tidyverse. Popular packages explored within include: ggplot2 and dplyr.

Data Source

The data used in this document is sourced from Kaggle. It is a consolidation of other well know “Food” related sources including: FAO Food Price Index, USDA Food Price OUtlook, World Bank, ICO World Indicator.

Global Basket Comparison

We will start with a macro view of 7 large cities spanning the world, comparing their basket price change over the past 6 months. We will use dplyr functions group_by() and summarise() to add up total base costs per month, per city.

Furthermore, we will use geom_line() in ggplot to create a simple line graph comparing prices over a period for each city.

# Calculate the total basket cost per city, per month (using dplyr)
basket_totals <- hub_data %>%
  group_by(City, Country, Month) %>%
  summarise(
    total_basket_usd = sum(Price_USD, na.rm = TRUE),
    .groups = "drop"
  )

Micro View

Packages used:

Dplyr | filter()

ggplot | geom_line(), facet_wrap()

To focus on a more granular level, we filter() the data to just 2 specific months and a few staples. We use facet wrap() function within ggplot to create multiple side by side plots. This is really helpful when too many levels of data clumped together would prove messy and ineffective to analyze.

# Further filter specific items
core_staples <- c("Milk_1L", "Bread_500g", "Eggs_12", "Bananas_1kg", "Apples_1kg")

# Filter the data
filtered_hub_data <- hub_data %>%
  filter(Item_Key %in% core_staples) %>%
  filter(Month_Name %in% c("October 2025", "March 2026")) 


ggplot(filtered_hub_data, aes(x = Price_USD, y = Item_Key)) +
  geom_line(aes(group = Item_Key), color = "grey80", linewidth = 1) +
  geom_point(aes(color = Month_Name), size = 3) +
  facet_wrap(~City, scales = "free_x") + 
  theme_minimal() +
  labs(
    title = "Price Shifts of Key Breakfast Staples",
    subtitle = "October 2025 vs March 2026",
    x = "Price (USD)",
    y = NULL,
    color = "Month"
  )

Interestingly from this analysis, it looks like Cairo and Toronto experienced significant increases in certain items (e.g., apples) while others like Milk remained relatively flat.

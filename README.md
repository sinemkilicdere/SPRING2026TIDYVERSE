I extended the vignette created by Kristoff Oliphant, which used the `msleep` dataset to analyze herbivores with high sleep duration.

### What I added
In my extension, I:
- Sorted the herbivores by total sleep using `arrange()`
- Calculated the average sleep time using `summarise()`
- Created a bar chart using `ggplot()` to visualize the results
